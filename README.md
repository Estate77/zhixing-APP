# 智行APP - 高保真原型

## 🌐 在线体验

**公开访问地址：** https://estate77.github.io/zhixing-APP/

## 📱 项目介绍

智行APP是一款智能汽车应用的高保真原型设计，包含以下核心功能模块：

- **首页**：车辆状态监控、快捷控制、活动推荐
- **社区**：车主交流、内容分享
- **智驾**：智能驾驶功能展示
- **远控**：远程车辆控制
- **我的**：个人中心、ADAS徽章、车辆信息

## 🎨 设计特点

- 采用 Minimalism + Glassmorphism 设计风格
- 配色方案：Electric Blue (#0066FF)、Deep Space Grey (#1A1F36)、Eco Green (#00D084)
- 响应式布局，支持移动端优先
- 包含AI用车助手、车主信息中心等下钻页面

## 🚀 技术栈

- 纯前端实现：HTML5 + CSS3 + JavaScript
- 单文件架构，无需构建工具
- 支持 viewport-fit=cover 和 safe-area 适配

## 📦 本地运行

直接在浏览器中打开 `index.html` 即可预览。
